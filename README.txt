###################
# cpdrawable V0.1 #
###################

this script will create sdpi, mdpi, xdpi... folders in /src/main/drawable/ and copy images from a folder to them 
/!\for the moment this script copy each images on each _dpi folders

##########
# usage: #
##########
1)copy a folder containing .png images to your android project's root path
3)open a terminal and make cpdrawable executable (chmod +x cpdrawable)
2)go to the project's root path and run ./<path to cpdrawable>/cpdrawable
3)enjoy
